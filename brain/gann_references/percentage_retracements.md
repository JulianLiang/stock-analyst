# 江恩百分比回檔位 (Gann Percentage Retracements)

## 理論背景
江恩認為市場趨勢的調整（回檔或反彈）通常會發生在特定的百分比位階。這些百分比是基於將一個完整的波動區間（高點到低點，或低點到高點）進行等分分割。

江恩特別強調「8 分法」與「3 分法」，其中最重要的是 **50% (4/8)** 位階，他認為這是所有支撐與壓力中最強大的一點。

## 核心位階表

| 分數 | 百分比 | 說明 |
| :--- | :--- | :--- |
| 1/8 | 12.5% | 初步調整位 |
| 2/8 | 25.0% | 重要支撐壓力位 |
| **1/3** | **33.3%** | 次要結構位 |
| 3/8 | 37.5% | 強勢回檔位 |
| **4/8** | **50.0%** | **終極平衡點 (最重要的轉折位)** |
| 5/8 | 62.5% | 深勢回檔位 |
| **2/3** | **66.6%** | 次要結構位 |
| 6/8 | 75.0% | 趨勢存亡線 |
| 7/8 | 87.5% | 最後防線 |
| 8/8 | 100.0% | 完全對稱轉折 |

## 計算方法
假設一段波段的高點為 `H`，低點為 `L`。

1. **區間幅度 (Range)** = `H - L`
2. **回檔目標價 (Retracement)** = `H - (Range * 百分比)`
3. **反彈目標價 (Rebound)** = `L + (Range * 百分比)`

## 實戰原則
- **50% 規則**：如果股價跌破 50% 且無法在三天內站回，通常代表原趨勢已經反轉，而非單純回檔。
- **重疊原則**：如果江恩百分比回檔位，剛好與江恩正方形 (Square of 9) 的基本角價位重合，該點位的效力會呈倍數增強，稱為「共振位」。
- **趨勢強度**：
    - 強勢趨勢通常只會回檔到 25% 或 37.5% 就繼續前進。
    - 弱勢趨勢則常回測 62.5% 甚至 75%。
